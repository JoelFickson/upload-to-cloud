# Cloud Send

⚠️ THIS IS NOT READY FOR USE YET

A package that enables you to upload files to cloud storage platforms.

### Currently Supported Platforms

- Google Cloud
- ⏳ Amazon S3
- ⏳ Azure Blob Storage
- ⏳ Digital Ocean Spaces
- ⏳ Cloudinary

## Installation

```bash
npm install cloud-send 
```
